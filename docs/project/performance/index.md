## 性能指标和优化目标

- Network 面板
  - waterfall 加载瀑布流，显示了各个阶段的时间
  - 存储 HAR 与重建性能信息
- Lighthouse 面板
  - First Contentful Paint
  - Speed Index: 速度指标 4s
- 重要测量指标(请求)
  - Speed Index: 速度指标 4s
  - TTFB
  - 页面加载时间
  - 首次渲染: 渐进渲染
- 响应
  - 交互动作的反馈时间
  - 帧率 FPS: 通过 ctrl+shift+p 后输入 frame，显示帧率
  - 异步请求的完成时间，加 loading 效果

指标只是指导作用，并不一定要达到，具体还是看需求。
