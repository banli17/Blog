# 图的基础

## 概念

图论(Graph Theory)：研究由点、边构成的一种数据模型的学科。

概念

- 点
- 边
  - 有向图
  - 无向图
- 权重
  - 有权图
  - 无权图
- 图的连通性：节点不一定都被连接起来
  ![](imgs/2020-11-08-16-47-41.png)
- 简单图(Simple Graph): 没有自环边和平行边
  - 自环边(self-loop)
  - 平行边(parallel-edges): 比如两个城市的道路
    ![](imgs/2020-11-08-16-47-57.png)
